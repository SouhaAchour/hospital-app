# Hospital Request Management App

A simple full-stack web application for managing medical exam requests.

##  Features

- User registration (Patient / Doctor)
- Secure authentication using JWT
- Role-based access control
- Doctors can create exam requests
- Patients can view their requests
- Session persistence using localStorage

##  Tech Stack

- Frontend: Angular
- Backend: Node.js / Express
- Database: MySQL
- Authentication: JWT

##  Project Structure

- /frontend → Angular application
- /backend → Express API
